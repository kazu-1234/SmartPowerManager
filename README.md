## 開発の背景と目的

夜中にPCで作業や動画視聴を行っている最中に寝落ちしてしまい、朝まで無駄にPCが稼働し続けてしまう「切り忘れ問題」は、電力の無駄遣いだけでなくPCパーツの寿命にも影響します。また、自宅で稼働させているサーバー用PCにおいては、システムの動作を安定させるために定期的な再起動が必要ですが、これを手動で毎回行うのは手間がかかります。

さらに、外出先から自宅のPCにアクセスしたい場面において、常にPCの電源を入れっぱなしにしておくのは非効率です。電気代を抑えつつ、必要な時だけ外部から電源を入れられる仕組みが求められていました。

本ソフトウェアは、これらの課題を解決し、PCのシャットダウンや再起動の自動化、および必要なタイミングでの遠隔起動を統合的に管理するために開発されました。

## システム概要

Windows PCの電源操作（シャットダウン・再起動）を柔軟にスケジュール管理する常駐型アプリケーションです。本アプリ単体での電源オフ管理に加え、Raspberry Pi Pico Wと連携することで、Wake on LAN (WoL) を利用したPCの遠隔起動機能を提供します。

## 主な機能（C# 版 v2.0.0）

- シャットダウン／再起動／起動スケジュール（毎日・毎週・一回限り・クイック）
- 監視オン／オフ（シャットダウンと再起動で独立）
- Pico W / GAS 連携（WoL 同期、3分前自動 WoL）
- タスクトレイ常駐、多重起動防止、ログオン時自動起動
- ライト／ダーク／システム連動テーマ
- GitHub Releases からのアップデート確認

## ビルド（C#）

```powershell
cd CSharp
dotnet publish SmartPowerManager.csproj -c Release -r win-x64 -p:Platform=x64 --self-contained true
```

## 動作環境

- **PC側**: Windows 10 / 11
- **マイコン側**: Raspberry Pi Pico W（WoLや外部連携を使用する場合）
