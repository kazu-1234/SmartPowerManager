# SmartPowerManager

PCのシャットダウンスケジュールを管理するWindowsアプリケーションです。

## 機能

- **毎日スケジュール**: 毎日同じ時刻にシャットダウン
- **毎週スケジュール**: 曜日ごとに個別設定可能
- **一回限りスケジュール**: 特定の日時にシャットダウン（最優先）
- **クイック設定**: 1, 3, 6, 9, 12時間後にシャットダウン
- **確認ダイアログ**: シャットダウン前に60秒のカウントダウン表示
- **MACアドレス表示**: Wake-on-LAN用（Raspberry Pi Pico W対応）

## 必要環境

- Windows 10/11
- Python 3.8以上

## インストール

```bash
git clone https://github.com/YOUR_USERNAME/SmartPowerManager.git
cd SmartPowerManager
```

## 使用方法

```bash
python SmartPowerManager_v1.4.0.pyw
```

または `.pyw` ファイルをダブルクリックして起動。
