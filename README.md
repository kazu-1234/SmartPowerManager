# SmartPowerManager

PCのシャットダウン/再起動スケジュールを管理し、Raspberry Pi Pico Wと連携してWoL（Wake on LAN）による遠隔起動を行うシステムです。

## 変更履歴 (v1.6.8)
- **バグ修正**: アップデート確認時に発生していたバージョン文字列のパースエラーを修正。
- **UIの最適化**: 設定タブにスクロールバーを追加し、マウスホイール操作に対応。不要な余白を削除し、レイアウトを安定化。
- **安定性の向上**: 設定保存時の排他制御（ロック）を導入し、データ破損を防止。

- **Web UIの刷新**: AJAXによるリロードなし更新、入力欄のゼロ埋め表示対応。
