# SmartPowerManager

![SmartPowerManager](images/main_overview.png)
*(※ここにメイン画面のスクリーンショットを配置してください)*

PCのシャットダウン・再起動スケジュールを柔軟に管理し、Raspberry Pi Pico Wと連携してWoL（Wake on LAN）による遠隔起動や、Google Apps Script (GAS) 経由でのLINEからの起動指示にも対応した統合電源管理システムです。

## 🌟 主な機能

### 💻 PC側アプリケーション (Windows用)
* **多彩なスケジュール管理機能**
    * **毎日・毎週設定**: 曜日ごとの個別設定や、毎日のルーティン実行が可能です。
    * **一回限り設定（最優先）**: 特定の日時に実行するスケジュールを予約できます。実行済みのタスクは記録・表示されます。
    * **クイック設定**: 現在時刻から「1, 3, 6, 9, 12時間後」の実行をワンクリックで手軽に予約できます。
* **シャットダウン / 再起動の分離**
    * シャットダウンと再起動で独立したタブを用意し、スケジュール設定・管理が可能です（排他制御により同時刻の競合を防止します）。
* **Raspberry Pi Pico W 連携（起動タブ）**
    * PCアプリからPico Wに対してPCの起動（WoL）スケジュール（毎日・毎週・一回限り）を送信し、自動で同期させることができます。
* **バックグラウンド動作とスタートアップ**
    * Windows起動時の自動実行（レジストリ登録）に対応しています。
    * アプリのウィンドウを閉じてもタスクトレイ（通知領域）に常駐し、バックグラウンドで監視を継続します。
* **安全な実行前確認**
    * スケジュール実行の60秒前に確認ダイアログ（カウントダウン）を表示し、直前でのキャンセル処理が可能です。画面オフ状態からのディスプレイ自動復帰にも対応しています。
* **自動アップデート機能**
    * GitHubリリースから最新バージョンを自動確認し、Rename-Swap方式による安全なアップデート機能（アンチウイルスソフトの誤検知やアクセス拒否を回避）を搭載しています。

### 🍓 マイコン側ファームウェア (Raspberry Pi Pico W)
* **WoL (Wake on LAN) 発信機能**
    * 指定したMACアドレスに対してマジックパケットを送信し、PCをネットワーク経由で起動します（デスクトップPC、サーバーPC等の個別指定も可能）。
* **Webブラウザ管理UI**
    * スマートフォンやPCのブラウザから直接アクセスし、各種スケジュールの追加・削除・確認が可能です。AJAX対応により画面リロードなしで快適に操作できます。
* **LINE連携 (GASポーリング)**
    * Google Apps Script (GAS) を介してLINE等からの指示をポーリング（マルチコアを活用しCore 1で非同期処理）し、「デスクトップPC起動」などの外部からのトリガーに即座に反応します。
* **NTP時刻同期と自律動作**
    * NTPサーバーによる正確な時刻同期を行い、対象のPCがシャットダウンしている間も単独でスケジューラーとして機能します。

## 📸 画面プレビュー・使用方法

### 1. シャットダウン / 再起動タブ
![シャットダウンタブ](images/shutdown_tab.png)
*(※ここにシャットダウン/再起動タブのスクリーンショットを配置してください)*
* 左側で「クイック設定（x時間後）」や「毎日のスケジュール」を設定し、右側で「毎週」「一回限り」のスケジュールを直感的に追加・管理できます。
* 一番下にはシステムログが表示され、いつどのような操作（キャンセル等含む）が行われたかがリアルタイムに記録されます。

### 2. 起動タブ (Pico W 同期)
![起動タブ](images/startup_tab.png)
*(※ここに起動タブのスクリーンショットを配置してください)*
* Pico Wに保存されるPCの自動起動スケジュールを設定します。
* 設定を追加・変更すると、自動的にPico Wへ送信（JSON形式でPOST通信）され、常に最新のスケジュールに同期されます。

### 3. 設定タブ
![設定タブ](images/settings_tab.png)
*(※ここに設定タブのスクリーンショットを配置してください)*
* **Windows起動設定**: OS起動時に自動実行する（タスクトレイ常駐）かどうかの切り替えを行います。
* **Raspberry Pi Pico W 設定**: Pico WのIPアドレスと、WoLで起動対象とするPCのMACアドレスを設定します。MACアドレスは現在のネットワークアダプタから自動スキャンしてリストから選択可能です。
* **動作モード**: デバッグモードを有効にすると、実際のシャットダウンや再起動処理を行わず、ログ出力のみで安全に動作確認ができます。

### 4. アップデートタブ
![アップデートタブ](images/update_tab.png)
*(※ここにアップデートタブのスクリーンショットを配置してください)*
* ボタン一つで最新バージョンを確認し、そのままアプリ内でダウンロードからファイル入れ替え、再起動までをワンストップで完結させます。

### 5. Pico W Web UI
![Pico W Web UI](images/picow_webui.png)
*(※ここにPico Wのブラウザ管理画面のスクリーンショットを配置してください)*
* Pico WのIPアドレスにブラウザからアクセスすると表示されます。PC版アプリと同様のスケジュール設定や、手動での即時WoL送信ボタン（デスクトップPC、サーバーPC、アプリ設定PCなど）が備わっています。

## ⚙️ システム要件・セットアップ

### PC側
* Windows 10 / 11
* （ソースから実行する場合）Python 3.x および必要なライブラリ (`pystray`, `Pillow` など)
* **ネットワーク**: 有線LAN接続（WoLを受信するため）、およびPCのマザーボード(BIOS/UEFI)とネットワークアダプタでの Wake on LAN の有効化設定が必要です。

### マイコン側
* Raspberry Pi Pico W
* Arduino IDE 環境
    * 必要なライブラリ: `WiFi`, `WebServer`, `NTPClient`, `WiFiClientSecure`
