## 開発の背景と目的
夜中にPCで作業や動画視聴を行っている最中に寝落ちしてしまい、朝まで無駄にPCが稼働し続けてしまう「切り忘れ問題」は、電力の無駄遣いだけでなくPCパーツの寿命にも影響します。
また、自宅で稼働させているサーバー用PCにおいては、システムの動作を安定させるために定期的な再起動が必要ですが、これを手動で毎回行うのは手間がかかります。

さらに、外出先から自宅のPCにアクセスしたい場面において、常にPCの電源を入れっぱなしにしておくのは非効率です。電気代を抑えつつ、必要な時だけ外部から電源を入れられる仕組みが求められていました。

本ソフトウェアは、これらの課題を解決し、PCのシャットダウンや再起動の自動化、および必要なタイミングでの遠隔起動を統合的に管理するために開発されました。

## システム概要
Windows PCの電源操作（シャットダウン・再起動）を柔軟にスケジュール管理する常駐型アプリケーションです。
本アプリ単体での電源オフ管理に加え、Raspberry Pi Pico Wと連携することで、Wake on LAN (WoL) を利用したPCの遠隔起動機能を提供します。これにより、外出先からでもスマートフォン等を経由して自宅のPCの電源を入れることが可能になります。

## 主な機能

### PC電源スケジュール管理
* **シャットダウンと再起動の完全分離**: タブで処理を分け、同時刻に設定した際の競合を防ぐ排他制御を実装しています。
* **クイックタイマー設定**: 現在時刻から「1時間後」「3時間後」など、数クリックでスケジュールを追加できます（寝落ち対策に最適です）。
* **柔軟なスケジュール設定**: 毎日、毎週（曜日指定）、一回限り（指定日時）の3種類からスケジュールを設定可能です。

### バックグラウンド動作の最適化
* **タスクトレイ常駐**: アプリケーションウィンドウを閉じても、タスクトレイに常駐してバックグラウンドで監視を継続します。
* **スタートアップ登録**: Windows起動時に自動で実行されるよう設定可能です。
* **安全機能**: スケジュール実行の60秒前にカウントダウンダイアログを表示します。意図しないタイミングでのシャットダウンを防ぐため、この画面から実行のキャンセルが可能です。

### Raspberry Pi Pico W 連携と Wake on LAN (WoL) 遠隔起動
* **PCの遠隔起動 (WoL)**: 待機電力の少ないRaspberry Pi Pico Wをローカルネットワーク内に常時稼働させておくことで、外部からの要求を受け取り、対象のPCへWake on LAN (WoL) のマジックパケットを送信して物理的に電源を入れます。自宅のサーバーPCやメインPCを任意のタイミングで起動できます。
* **スケジュールの自動同期**: PCアプリ側で設定した「起動スケジュール」を、自動的にPico W側へ送信・保存させます。PC本体がシャットダウンしている状態でも、Pico Wがスケジュールに従って自動でPCを起動させます。
* **Webブラウザからの手動起動**: Pico WのIPアドレスにスマートフォンなどのブラウザからアクセスし、管理画面（Web UI）からボタン一つで対象のPCを即座に起動させることができます。

Google Apps Script (GAS) と連携し、LINEなどの外部サービスからPCを起動させることができます。
例えば、LINEのトーク画面で「デスクトップPC起動」や「サーバーPC起動」とメッセージを送信するだけで、Pico Wがそれを検知して対象のPCを起動します。操作後にはLINE上に「起動しました」という応答メッセージが返るため、外出先からでも確実に処理が実行されたことを手元で確認できます。

### 自動アップデート
* GitHubから最新のリリース情報を取得し、アプリ単体でアップデートを完結させます。実行中のファイルを安全に置き換えるRename-Swap方式を採用しているため、セキュリティソフトの誤検知やアクセス拒否エラーを防ぎます。

## 動作環境

* **PC側**: Windows 10 / 11
* **マイコン側**: Raspberry Pi Pico W (WoLや外部連携を使用する場合)
  * 開発環境: Arduino IDE
